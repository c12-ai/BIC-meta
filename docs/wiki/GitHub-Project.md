# GitHub Project

The BIC GitHub Project is the operational board for coordinating repos, docs, issues, PRs, and release-relevant commits.

Project URL:

https://github.com/orgs/c12-ai/projects/9

## Fields

| Field | Purpose |
| --- | --- |
| `Status` | Todo, In Progress, Done. |
| `Area` | Repos, Wiki, Documents, Issues, PRs, Commits. |
| `Repo Scope` | Owning repo or Cross-repo. |

## Project Item Types

- Real GitHub Issues for actionable repo work.
- Real GitHub Pull Requests for review and merge tracking.
- Draft items for wiki, document, repo-map, and commit-tracking work.

## Current Visibility Note

The `BIC-meta` repository is public. The org-level Project visibility may require organization admin permission to make public.
