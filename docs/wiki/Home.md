# BIC Wiki

This wiki is the shared entry point for BIC project coordination.

## Primary Links

- Root meta repo: https://github.com/c12-ai/BIC-meta
- GitHub Project: https://github.com/orgs/c12-ai/projects/9
- Production PRD: https://github.com/c12-ai/BIC-meta/blob/main/Production-PRD.md

## Managed Areas

- Repositories: root meta repo and child repositories.
- Documents: Production PRD, Project PRDs, SOP index, and architecture notes.
- Wiki: durable project knowledge and onboarding notes.
- Issues: requirements, bugs, chores, and follow-ups.
- Pull requests: review state, merge readiness, and cross-repo coordination.
- Commits: release-relevant or cross-repo changes that need visibility.
