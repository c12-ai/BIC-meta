# Use three reuse zones for cross-domain Agent workflows

The common substrate is divided into Agent Foundation for domain-neutral execution mechanics, an Experiment Workflow Kit for experimental-product behavior proven reusable across domains, and Domain Packs for irreducible scientific semantics. These are reuse zones within L1-L4 rather than new architectural layers: Foundation remains a governed execution kernel, the Kit supplies layer-aligned L2/L3 components without owning persistence or execution authority, and each Pack composes those capabilities instead of returning an arbitrary compiled graph or rebuilding model, tool, context, stream, lifecycle, and experiment-progression machinery. Safety and authority invariants enter the neutral core immediately; other workflow behavior is promoted into the Kit only after Chemistry and Biology demonstrate identical semantics and the extraction removes real duplication without domain conditionals.
