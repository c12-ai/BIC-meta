# Validate Domain Pack contracts before legacy L3 retirement

BIC will use production-shaped Chemistry slices to make the provisional Agent Foundation public SPI, Experiment Workflow Template/Kit contracts, and Domain Pack contract concrete, but it will not treat one domain as proof of portability. A minimal non-production Biology slice must exercise the same lifecycle through production-shaped ports and may reshape that exact freeze set. After the set freezes, every migrated Chemistry slice must pass again, whether or not a Biology-driven change appeared to affect it. Only after Biology passes, the contracts freeze, Chemistry is re-proven, and every unfinished old-path workflow on the bench and at field sites has completed or been explicitly reset may the default switch once and legacy routing/code be deleted. The validation does not authorize Portal, Lab Service, shared-types, or production Biology changes.
