# Migrate L3 through vertical slices

The L3 redesign will use independently verifiable end-to-end slices rather than a horizontally complete platform-first build or a big-bang cutover. The first new path is a production-shaped internal run of the complete baseline Chemistry happy path from Objective through the deterministic ELN Summary Document, including applicable confirmation, Lab round trips, Session Event projection, and replay; happy-path success alone does not admit arbitrary production workflows. Remaining Chemistry behavior migrates incrementally behind internal or disabled routing. A trusted L2 release policy selects an exact legacy or new Behavior Target only when the immutable Workflow Behavior Binding commits, and every later Turn and Step stays on that target, with no mutable deployment mapping, unsupported-branch fallback, effectful shadow run, dual write, or second external-command authority. Production eligibility requires all behavior reachable by the cohort to pass, the non-production Biology slice to pass, the Agent Foundation public SPI, Experiment Workflow Template/Kit contracts, and Domain Pack contract to freeze, and every migrated Chemistry slice to pass again against that frozen set. Legacy routing and code are deleted only after parity, no-live-binding, compatibility, and rollback gates pass.
