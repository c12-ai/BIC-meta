# Preserve L1-L4 as responsibility boundaries, not topology

BIC will retain L1-L4 as its stable technical responsibility and dependency model: L1 adapts protocols, L2 hosts cross-turn coordination and deterministic workflow change, L3 performs turn-scoped Agent execution and produces typed intents or proposals, and L4 implements technical adapters. Current file placement, graph topology, and Chemistry specialist wiring are not protected and may be restructured. Dependency violations must fail CI because documentation-only boundaries have already drifted in the live implementation.
